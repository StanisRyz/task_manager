# AGENTS

## Project goal
Собрать офлайн-менеджер задач (TODO) на Flutter с русским интерфейсом.

## Commands to run
- `flutter pub get`
- `flutter analyze`
- `flutter test`

## Coding style
- Небольшие файлы и понятные имена.
- Полная null-safety.
- Без мёртвого кода.
- Форматировать через `dart format`.

## Testing expectation
- Должен проходить хотя бы один widget-тест.
